- GasHound LI-800
- Cubic CU-1106-C

- Cubic CU-1106SL-NS
- Cubic CU-1106SL-N
- MH-Z16 w/ I2C
- MH-Z16 w/ PWM
- MH-Z14 A
- SEN-0219
